📱 Visão Geral

A To-Do List (Lista de Tarefas) é uma aplicação desenvolvida para auxiliar na organização pessoal, permitindo o gerenciamento simples e eficiente de tarefas do dia a dia.

O sistema funciona via linha de comando (CLI), oferecendo uma experiência rápida e direta, ideal para praticar lógica de programação e estruturação de código.

🎯 Objetivo do Projeto

Este projeto tem como finalidade reforçar conceitos fundamentais de desenvolvimento, além de simular uma solução prática para organização de atividades.

Entre os principais objetivos estão:

Melhorar a lógica de programação

Trabalhar com manipulação de listas

Organizar código de forma clara e reutilizável

Simular persistência de dados

⚙️ Funcionalidades

➕ Adicionar Tarefa

Permite criar novas tarefas com descrição personalizada.

❌ Remover Tarefa

Remove tarefas da lista de forma simples e rápida.

✅ Marcar como Concluída

Atualiza o status da tarefa para concluída.

🔍 Filtrar Tarefas

Visualização separada entre:
Tarefas pendentes
Tarefas concluídas

💾 Persistência de Dados (Diferencial)

O sistema permite salvar as tarefas em arquivo local, simulando o funcionamento de um banco de dados.

📂 Armazenamento em arquivo (ex: .json ou .txt)

🔄 Carregamento automático ao iniciar o sistema

💡 Garante que as tarefas não sejam perdidas ao encerrar o programa

🧠 Conceitos Aplicados

Este projeto foi desenvolvido com foco no aprendizado de:

📚 Listas e Estruturas de Dados
Armazenamento e manipulação de tarefas

🔁 Funções
Organização das operações do sistema

🧩 Modularização
Separação de responsabilidades no código

💾 Manipulação de Arquivos
Leitura e escrita de dados
